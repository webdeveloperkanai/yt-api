API Endpoints

⭐ Main Feature

GET /api/stream

DIRECT VIDEO & AUDIO STREAMING API: Use this endpoint to instantly play YouTube videos or audio in your own application without proxying or downloading the entire file first.

Exact API Usage (Copy & Paste)

// To stream Video:
http://localhost:3000/api/stream?url=YOUTUBE_URL&res=720

In your HTML Project

<video controls src="http://localhost:3000/api/stream?url=YOUTUBE_URL&res=720"></video>
Parameter Type Description
url string Full YouTube URL (Required)
type string 'video' (Default) or 'audio'
res string Resolution e.g., '720', '1080', '360'

GET/api/stream?type=audio

Audio Only Stream: Instantly stream the best available audio directly.

<audio controls src="http://localhost:3000/api/stream?url=YOUTUBE_URL&type=audio"></audio>

GET/api/search

Search for videos.

Parameter Type Description
q string Search query (Required)
limit number Max results (Default 10, Max 30)

GET/api/info

Get detailed metadata for a URL.

Parameter Type Description
url string Full URL (Required)

GET/api/links

Extract available direct format links bypass stream proxy.

Parameter Type Description
url string Full URL (Required)

GET/api/raw

Raw YouTube CDN URLs: Returns direct googlevideo.com stream URLs for every available video and audio format — no proxy, straight from YouTube's servers.

Parameter Type Description
url string Full YouTube URL (Required)

Response shape:

{\n rawVideo: [{ quality, resolution, fps, ext, vcodec, acodec, size, url }],\n rawAudio: [{ quality, ext, acodec, abr, size, url }]\n}

GET/api/download

Force direct file download (Sets Content-Disposition attachment).

Parameter Type Description
url string Full URL (Required)
type string 'video' or 'audio'
